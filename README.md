# riceleafdiseasesproj

An end-to-end computer vision and deep learning project designed to automate the detection of diseases in rice leaves. This project utilizes a custom Convolutional Neural Network (CNN) architecture trained on image data to accurately classify various crop diseases, helping farmers protect their yields.

# Features
Custom CNN Architecture: Engineered from scratch using TensorFlow and Keras.
Image Preprocessing: Includes data augmentation, rescaling, and resizing pipelines.
Interactive Notebook: Built and optimized originally in Google Colab for easy reproduction.
